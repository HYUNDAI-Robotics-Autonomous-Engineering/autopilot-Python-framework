Mouse_List¶

class rpilot.core.gui.Mouse_List(mice=None, drop_fn=None)[source]¶

Bases: PySide.QtGui.QListWidget

A trivial modification of QListWidget that updates pilots when an item in the list is dragged to another location.

Should not be initialized except by Control_Panel .

Variables:
  • mice (list) – A list of mice ID’s passed by Control_Panel
  • drop_fn (Control_Panel.update_db()) – called on a drop event
Parameters:
  • mice – see mice. Can be None for an empty list
  • drop_fn – see drop_fn(). Passed from Control_Panel
