    • Pilot_Panel¶

    class rpilot.core.gui.Pilot_Panel(pilot=None, mouse_list=None, start_fn=None, create_fn=None)[source]¶

    Bases: PySide.QtGui.QWidget

    A little panel with

    • the name of a pilot,
    • A Pilot_Button to start and stop the task
    • Add and remove buttons to create_mouse() and Pilot_Panel.remove_mouse()

    Note

    This class should not be instantiated except by Control_Panel

    Variables:
    • layout (QtGui.QGridLayout) – Layout for UI elements
    • button (Pilot_Button) – button used to control a pilot
    Parameters:
    • pilot (str) – The name of the pilot this panel controls
    • mouse_list (Subject_List) – The Subject_List we control
    • start_fn (toggle_start()) – Passed by Control_Panel
    • create_fn (Control_Panel.create_mouse()) – Passed by Control_Panel

    Methods Summary
