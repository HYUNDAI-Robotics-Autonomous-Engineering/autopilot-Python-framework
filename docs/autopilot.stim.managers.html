class autopilot.stim.managers.Stim_Manager(stim=None)[source]Â¶

Bases: object

Yield sounds according to some set of rules.

Currently implemented:

  • correction trials - If a subject continually answers to one side incorrectly, keep
    the correct answer on the other side until they answer in that direction
  • bias correction - above some bias threshold, skew the correct answers to the less-responded side

Variables:
  • responses (collections.deque) â History of prior responses
  • targets (collections.deque) â History of prior targets.
Parameters:
  • mode â

    One of the following:

    • âthresholded linearâ : above some threshold, do linear bias correction
      eg. if response rate 65% left, make correct be right 65% of the time
  • thresh (float) â threshold above chance, ie. 0.2 means has to be 70% biased in window
  • window (int) â number of trials to calculate bias over
