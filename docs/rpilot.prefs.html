    • prefs¶

    Module to hold module-global variables as preferences.

    Warning

    DO NOT hardcode prefs here.

    A prefs.json file should be generated with an appropriate rpilot.setup routine

    Before importing any other RPilot module,

    Examples

    from rpilot import prefs prefs_file = ‘/usr/rpilot/prefs.json’ # or some .json prefs file prefs.init(prefs_file)

    And to add a pref

    Examples

    prefs.add(‘PARAM’, ‘VALUE”)

    rpilot.prefs.prefdict = {'AGENT': 'docs', 'AUDIOSERVER': 'docs'}¶

    stores a dictionary of preferences that mirrors the global variables.
